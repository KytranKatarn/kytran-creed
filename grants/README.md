## How to Use These Templates

1. **Start with `narrative-template.md`** — copy it, replace all `[BRACKETED]` placeholders with funder-specific content.
2. **Attach `budget-template.md`** — fill in personnel `[AMOUNT]` fields to match the grant ceiling. Keep overhead at 15%.
3. **Reference `impact-framework.md`** — paste the metrics table into the evaluation/impact section of each application.
4. **Customize the alignment section** — the final section of the narrative template is funder-specific. Write 1–2 paragraphs explaining why C.R.E.E.D.'s work matches that funder's priorities.
